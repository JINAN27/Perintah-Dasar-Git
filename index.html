Inisialisasi Git

Gunakan perintah "git init"

untuk config gunakan perintah "git config --list"

untuk melihat status gunakan "git status"

untuk memasukkan ke staging area gunakan "git add "

untuk memberikan pesan dan sudah sepakat gunakan "git commit"

apabila ingin mengclone repositori gunakan "git clone"

untuk terhubung dengan github gunakan "git remote add origin "url/ssh" "

untuk mengirim file dari git local ke github gunakan "git push"

untukk mengambil commit dari server gunakan "git pull"
